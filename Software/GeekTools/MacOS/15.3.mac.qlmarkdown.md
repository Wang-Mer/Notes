## QLMarkdown 安装使用教程  

​    

​    

### 1 下载  

Github 地址: [https://github.com/toland/qlmarkdown](https://github.com/toland/qlmarkdown "https://github.com/toland/qlmarkdown")    

​    

### 2 安装    

使用 `homebrew` 安装 `brew cask install qlmarkdown`  

​    

### 3 使用  

选中文件，点击「空格」即可预览文件
