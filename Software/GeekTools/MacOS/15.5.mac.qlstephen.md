## QLStephen 安装使用教程  

​    

​

Github 地址: [https://github.com/whomwah/qlstephen](https://github.com/whomwah/qlstephen "https://github.com/whomwah/qlstephen")

使用 `homebrew` 安装 `brew cask install qlstephen`

选中文件，点击「空格」即可预览文件
